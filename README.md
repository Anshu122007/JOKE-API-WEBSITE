# Joke API Demo

## Description
This is a simple web application that fetches jokes from the JokeAPI. Users can select a category, click a button to get a joke, and interact with the joke by clicking it for a new one. The app also handles loading states and errors for a smoother user experience.

## Features
- Select joke categories.
- Get a random joke based on the selected category.
- Click the joke to fetch a new one.
- Error handling and loading states.
- Mobile-friendly design.
